# Spatial Memory Operations

| Field | Value |
|---|---|
| Page ID | SM-OPERATIONS |
| Confluence parent | SM-ROOT |
| Page role | Operational handoff parent |
| Child page ID | SM-OPERATIONS-HANDOFF |
| Child canonical source | Repository `HANDOFF.md` |
| Status | Active preparation |
| Last reviewed | 2026-07-12 |

이 페이지는 Confluence의 operational parent다. 실제 실행, 승인, 취소, artifact
handoff 절차는 repository root의 `HANDOFF.md` 한 곳에서만 유지하고 Page ID
`SM-OPERATIONS-HANDOFF`로 가져온다. 이 디렉터리에 runbook 사본을 만들지 않는다.

실행 전에는 [Current Status](../status.md), [Architecture](../architecture.md),
[Experiments](../experiments/README.md)를 먼저 확인한다.
